# OralCare Clone Project

Professional dental services website clone.

## Getting Started

### Prerequisites
- Node.js & npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used
- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Deployment
Recommended platforms for deployment:
- [Vercel](https://vercel.com)
- [Netlify](https://www.netlify.com)
